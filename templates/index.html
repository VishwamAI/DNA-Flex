DNA-Flex

Getting Started

DNA-Flex provides powerful tools for DNA sequence analysis, structure prediction, and AI-powered genomics research.

Authentication

To use the API endpoints, you need to authenticate first:

  1. Get an access token by making a POST request to /token with your credentials
  2. Include the token in the Authorization header: Bearer your-token-here

Example using curl:

curl -X POST "http://localhost:8000/token" -d "username=your-username&password=your-password"
